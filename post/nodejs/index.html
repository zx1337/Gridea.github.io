nodejs

2022-07-21 3 min read

npm，即 Node Package Manager，是管理项目用到的js库（js文件）
nodejs中包含了npm，需要安装nodejs，并且配置环境
VUE框架，前后端分离
前台代码和后台代码在一个项目中，它们被部署在同一个服务器中。（Servlet+JSP)
前台代码在前台服务器中，后台代码在后台服务器，两个项目
效率高，解耦合
VUE这个框架简化了JS的操作，把JS封装成一个VM对象（试图＋数据模型）

  • 百度nodejs

  • http://nodejs.cn/ 官网

  • 下载长期支持版

  • 解压安装

  • 配置环境变量
    右键我的电脑属性或run:sysdm.cpl
    增加:D:\nodejs\node_global保存下载的程序包路径
    增加:D:\nodejs
    cmd：node -v 和 npm -v 命令验证版本信息，确定安全完成

  • D:\nodejs目录下新建目录node_cache 和 node_global

    cmd:npm config set prefix "D:\nodejs\node_global"
    npm config set cache "D:\nodejs\node_cache"

  • npm WARN
    显示npm WARN config global --global, --local are deprecated. Use --location=global instead.
    修改npm文件:prefix -g 为 prefix --location=global
    修改npm.cmd、npx、npx.cmd 文件同上

  • 镜像站点
    cmd:npm config set registry http://registry.npm.taobao.org(安装package容易报错)
    原始镜像：cmd:npm config set registry http://registry.npmjs.org(这个安装package不容易报错)

  • 查看配置信息
    cmd:npm config list 关注用户目录下.npmrc文件

  • 安装VUE
    npm install vue -g 其中 install可以简写为i,-g是下载到global目录，执行命令并查看
    D:\nodejs\node_global\node_modules\vue

  • 安装路由模块
    cmd:npm install vue-router -g

  • 安装脚手架
    建工程用的模板
    npm install vue-cli -g
    vue-cli内置两个工具，webpack和webpack-simple,前者专业，后者简单
